# Научиться учиться

## О проекте 
Это учебный проект в рамках курса Веб-разработчик от ЯндексПрактикума, где я сверстал, _обучающе-мотивационный сайт_ по шаблону. Который помогает людям начать свой путь к развитию.

### Технологий:

- HTML5
- CSS3
  - Flexbox
  - Grid Layout
  - Animations
  - Iframe
- БЭМ Nested
